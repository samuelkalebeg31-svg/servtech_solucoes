ServTech Soluções — Sistema Tkinter (Protótipo Didático)
=======================================================
Este projeto demonstra um fluxo simples de autenticação e gerenciamento
de ordens de serviço em Python com Tkinter e SQLite.

Como executar
-------------
1) Certifique-se de ter o Python instalado.
2) Em um terminal, navegue até a pasta do projeto e rode:
   python app.py

Estrutura
---------
app.py         -> Interface Tkinter (Login + Tela de Ordens de Serviço)
db.py          -> Conexão e inicialização do banco SQLite
repository.py  -> Acesso a dados (usuários e ordens)
validators.py  -> Funções de validação (protótipo)
